# TChat
Advanced chat plugin | JSON

# Features
- JSON config
- Multiple formats
- Chat clear command
- Ban words
- PlaceholderAPI support
- PlaceholderAPI expansion
- AntiBot

# Commands
- /chat reload
- /chatclear

# Permissions
- tchat.reload
- tchat.group.(group)
- tchat.admin.chatclear

# Dependencies
- PlaceholderAPI

# Placeholders
- %tchat_prefix%
- %tchat_suffix%
