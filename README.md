# TChat
Advanced chat plugin | JSON

# Features
- JSON config
- Multiple formats
- Chat clear command
- Ban words
- PlaceholderAPI support
- PlaceholderAPI expansion
- AntiBot
- AntiCap
- AntiFlood
- AntiSpam
- AntiAdvertising
- BanCommands
- MsgFormatter

# Commands
- /chat reload
- /chatclear
- /chat version
- /msg
- /cc
- /tell
- /whisper
- /ewhisper
- /pm
- /emsg

# Permissions
- tchat.reload
- tchat.group.(group)
- tchat.admin.chatclear
- tchat.version
- tchat.bypass.bannedwords

# Dependencies
- PlaceholderAPI

# Placeholders
- %tchat_prefix%
- %tchat_suffix%
